# 动画效果
> github: https://github.com/daneden/animate.css

## 使用方法
1. 先引用 `animate.css` 文件。
2. 添加类名：`animated 动画效果名称`。
	如：`<p class="animated bounce"></p>`
	> 默认只执行一次。如果想无限循环需要添加 `infinite` 
	如：`<p class="animated infinite bounce"></p>`
